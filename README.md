# CCNA 2 - Switching, Routing y Wireless Essentials

Repositorio de laboratorios prácticos del curso **CCNA 2** de Cisco Networking Academy. Cada módulo incluye topologías de referencia, tablas de direccionamiento y configuraciones paso a paso listas para replicar en **Cisco Packet Tracer** o equipos reales.

---

## 📂 Contenido

| # | Módulo | Descripción breve |
|---|--------|-------------------|
| 01 | Configuración Básica CLI | Hostname, contraseñas, banner, consola y guardado en NVRAM |
| 02 | Enrutamiento Estático | Rutas estáticas entre redes LAN interconectadas por enlaces seriales |
| 03 | VLAN | Creación de VLANs, asignación de puertos, trunk e InterVLAN Routing |
| 04 | Telnet y SSH | Acceso remoto seguro a routers y switches |
| 05 | Servidor DHCP, Web, DNS y Correo | Servicios de red sobre un servidor centralizado |
| 06 | Seguridad de Puertos en Switch | Port security, sticky MAC y políticas de violación |
| 07 | EtherChannel | Agregación de enlaces entre switches con LACP/PAgP |
| 08 | Protocolo OSPF | Enrutamiento dinámico de estado de enlace |
| 09 | Protocolo HSRP | Redundancia de gateway con Hot Standby Router Protocol |
| 10 | DHCP | Configuración de servidor y relay DHCP en routers Cisco |
| 11 | Protocolo Spanning Tree | STP, RSTP y prevención de bucles en redes conmutadas |

---

## 🛠️ Herramientas

- **Cisco Packet Tracer** (simulación)
- Equipos reales: routers **Cisco 1841**, switches **Cisco 2960-24TT**

---

## 🚀 Cómo usar este repositorio

Cada módulo contiene un PDF con la topología, tabla de direccionamiento y los comandos CLI necesarios. Para replicar un laboratorio:

1. Abrí Packet Tracer y armá la topología según el diagrama del módulo.
2. Seguí los pasos del PDF en orden.
3. Verificá la conectividad con los comandos de comprobación al final de cada sección.
